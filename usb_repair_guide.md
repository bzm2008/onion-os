# U盘修复指南

## 当前状态
U盘显示 "No Media"（无媒体），大小为 0

## 尝试以下方法（按顺序）：

### 方法1：重新插拔
1. 拔掉U盘
2. 等待10秒
3. 插入另一个USB接口（最好是USB 2.0）
4. 检查是否识别

### 方法2：设备管理器修复
1. 右键"此电脑" → "管理" → "设备管理器"
2. 展开"磁盘驱动器"
3. 找到你的U盘（ChipsBnk Flash Disk）
4. 右键 → "卸载设备"
5. 拔掉U盘，重新插入
6. Windows会自动重新安装驱动

### 方法3：DiskGenius修复（推荐）
下载 DiskGenius 免费版：
- 官网：https://www.diskgenius.cn/
- 或搜索 "DiskGenius 免费版"

步骤：
1. 打开 DiskGenius
2. 找到U盘（可能显示为"未初始化"）
3. 右键U盘 → "建立新分区"
4. 选择 FAT32 格式
5. 保存更改

### 方法4：量产工具（最后手段）
如果以上都失败，需要找U盘对应的量产工具：
1. 用 ChipGenius 查看U盘主控芯片型号
2. 搜索对应型号的量产工具
3. 重新量产U盘

## 如果都无法修复
U盘可能物理损坏，建议：
1. 换一个U盘
2. 或者使用 DVD 刻录 ISO
3. 或者在虚拟机中测试 Onion OS
